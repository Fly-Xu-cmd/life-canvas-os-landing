# Life Canvas OS 官网

这是 Life Canvas OS 项目的官方网站，部署在 GitHub Pages。

## 项目介绍

Life Canvas OS 是一款基于八维生命平衡模型的 AI 驱动个人成长桌面操作系统。

### 核心特性

- **AI 智能助手** - 自然语言交互，多轮对话记忆上下文
- **八维系统** - 饮食、运动、认知、产出、梦想、财务、社交、环境
- **隐私优先** - 所有数据本地存储

## 本地预览

直接在浏览器中打开 `index.html` 即可预览。

## 部署

本网站通过 GitHub Actions 自动部署到 GitHub Pages。
